> 这是对之前一个项目的升级 [旧版项目地址](https://github.com/wkunzhi/rbac-stark-crm)
包含可拆卸 权限组件rbac 与 stark插件 可以单独取出代码并运用到任意后端开发项目中。 该套CRM系统就是用rbac组件+stark组件开发的实例项目

# 开发环境
- Django 2.2.1
- Python 3.6

# 目录
- rbac **权限与菜单组件**
- stark **路由组件**
- web **业务逻辑**
- static **静态文件**
